# hashicorp-japan
Welcome to hashicorp-japan GitHub Organization!

HashiCorp Japan の GitHub Organization へようこそ！

ここでは、HashiCorp プロダクトに関するワークショップマテリアルや便利なツール群などを日本語情報で共有しています。

## ワークショップマテリアルについて

一部のワークショップマテリアルでは、HCP（HashiCorp Cloud Platform）および HCP Terraform のアカウントを利用することがあります。\
各アカウントのサインナップ手順については、[HCP Terraform, HCP のサインナップ方法](./account-signup.md) を参照するようにしてください。

## Contributing について

各リポジトリ内のコードやドキュメントについては、HashiCorp Japan メンバー間でのレビューを行っていますが、もしドキュメントエラーやツールのバグなどを発見された方がいましたら、ぜひ Issue や Pull Request を作成してください！\
（なお、Pull Request に対する対応については、ベストエフォートとなる点については留意いただけると幸いです）\
Issue および Pull Request 作成時のガイドラインについては、各リポジトリの CONTRIBUTING.md を参照するようにしてください。
