# HCP Terraform, HCP のサインナップ方法
ここでは、HCP Terraform と HCP (HashiCorp Cloud Platform, ここでは HCP Terraform 以外の HCP サービス)のサインナップ(ユーザ登録)方法を説明します。


## HCP Terraform のサインナップ
+ HCP Terraform の[トップページ](https://app.terraform.io/signup/account) にアクセスしてください。
+ "Username”,”Email”,”Password” を入力してください。2つのチェックボックスをチェックし、“Create account”をクリックしてください。
+ この画面が表示されるので、指定したメールアドレスの受信ボックスを確認してください。メールが届いていなかったらメールフィルタ(迷惑メールフォルダなど)を確認したり、この画面の"Resend Confirmation Link"をクリックしてください。
+ 指定したメールアドレスの受信ボックスに、こちらのようなメールが来ていると思います。メール文中の”clicking on this link”をクリックしてください。
+ この画面に推移するので、”Organization name”と”Email address”を入力し、”Create organization”をクリックします。これで登録は完了です。

## HCP (HashiCorp Cloud Platform, ここでは HCP Terraform 以外の HCP サービス)のサインナップ
+ HashiCorp Cloud Platform の[トップページ](https://portal.cloud.hashicorp.com/orgs/create) にアクセスし、”Create an account”をクリックください。
+ ”Email address”に任意のメールアドレスを入力し、”Continue”をクリックしてください。
+ 条件に合うパスワードを設定してください。
+ Service agreementsの内容を確認し、チェックボックス全てをチェックし、”Continue”をクリックしてください。
+ この画面が表示されるので、指定したメールアドレスの受信ボックスを確認してください。メールが届いていなかったらメールフィルタ(迷惑メールフォルダなど)を確認したり、この画面の"Resend email verification"をクリックしてください。
+ 指定したメールアドレスの受信ボックスに、こちらのようなメールが来ていると思います。メール文中の”Verify”をクリックしてください。
+ ・”Organization name”を入力し、"Create Organization"をクリックしてください。
